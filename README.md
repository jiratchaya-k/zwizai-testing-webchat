# ZWIZ.AI Testing Webchat

A lightweight web chat application built with Next.js and TypeScript that integrates with the LINE Messaging API to display conversations from a LINE Official Account (LINE OA) through a web-based interface. Developed as part of a testing submission for ZWIZ.AI.

## Demo

https://zwizai-testing-webchat.vercel.app

## Features

- Web chat interface for LINE Official Account
- View chat conversations from LINE users
- Fetch and display chat history dynamically
- State management for chat data
- Simple testing environment for webhook message handling

## Tech Stack

- Next.js
- React
- TypeScript
- Tailwind CSS
- TanStack Query (React Query)
- Zustand
- LINE Messaging API

## Getting Started

1. Clone the repository

2. Install dependencies

    `pnpm install`

3. Run the development server

    `pnpm dev`

4. Open the application
   Open http://localhost:3000
   in your browser.

## Deployment

This project is deployed on Vercel for testing purposes.

Webhook events from LINE Messaging API are handled via Next.js API routes.
